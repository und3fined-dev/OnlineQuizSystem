# Database

This folder contains SQL scripts for the Online Quiz System.

`QuizSysDB.sql` → contains `CREATE TABLE` commands.
`DataDB.sql` → contains sample data inserts.

# How to use
1. Open MySQL Workbench.
2. Create a new schema (e.g. `QuizSysDB`).
3. Run `QuizSysDB.sql` first to create tables.
4. Run `DataDB.sql` to insert sample data.
